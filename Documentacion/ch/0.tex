\begin{abstract}
A lo largo de esta guía se pretende mostrar cómo instalar desde cero un servidor
web con Nginx, realizando las operaciones necesarias para lograr el mayor
rendimiento y seguridad posibles con programas tales como php-fpm, APC, y el
módulo pagespeed de Google para optimizar los recursos web.
\end{abstract}
